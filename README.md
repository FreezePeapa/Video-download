# Youtube-download
使用操作:  
1.下載YTdownload_GUI.exe及ffmpeg.zip  
2.解壓縮ffmpeg.zip後，將ffmpeg.exe檔案放置到以下路徑:  
OS(C:)>Windows>System32>  
3.執行YTdownload_dlp.exe，可以開始下載音樂囉
